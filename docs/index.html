⇅   NoVario View on GitHub

USB OTG connect barometric sensor to serve as emergency vario connected to cellphone. Get rid of horrifying empty-batteries-in-vario situations! This board has simple MCU which is USB capable and one of the most precise barometric sensor.

  • PCB and board schematic is available online at EasyEDA
  • Firmware is generated with help of Atmel START and PlatformIO
  • To upload firmware 3-pin SWD interface is used (personally JLink is used)
